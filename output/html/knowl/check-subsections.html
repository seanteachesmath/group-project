Checkpoint 1.4.9. Divisions.

Read more about subsections and documents divisions here.

The three basic divisions inside of each pretext book are going to be the <chapter>, <section>, and <subsection>. In my experience, when using the <subsection> tag, a <introduction> and a <conclusion> must be included before and after any subsection in order for the compiler not to complain.
